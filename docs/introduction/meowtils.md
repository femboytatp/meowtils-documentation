# Meowtils

Meowtils is a utility mod with features that are both centered around specific gamemodes and servers but it also contains general QOL enchancements that are useful anywhere.

---

!!! Warning

    **All potentially disallowed features can be fully disabled, and they are already disabled by default. This means Meowtils itself should not be a disallowed mod on any servers.**

Meowtils is **NOT** a cheat client, it may however contain _certain_ features that may not be allowed on most servers as they provide a potentially unfair advantage (e.g 1.7 hitreg on 1.8 servers or certain render modules).

_While modules are specifically made to not cause any detection issues regardless by simulating vanilla game behaviour, certain modules do perform actions for you that, if used on strict anticheats may end up causing issues. An example of this would be **AutoChest** with a low delay or certain **DelayRemover** options._

**Risky modules like these are located in the _Advanced_ category.**

---

## Installation
1. Download and install [Forge](https://files.minecraftforge.net/net/minecraftforge/forge/index_1.8.9.html) and then select it as your installation in your launcher, or install Forge in any third party launcher.
2. Launch your game once to create a "mods" folder in your instance directory, or do so manually.
3. Drag the Meowtils.jar file into the mods folder.
4. Launch your game again, it should now be loaded.

!!! Note

    The default GUI keybind is Right Shift, if you are unable to open it you can do /meowtilsgui as a command in chat, you can also rebind this by middle clicking the **GUI** module.
