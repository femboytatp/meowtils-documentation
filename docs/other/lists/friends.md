# **Friends**

This allows you to add players to a friends list. Friended players will have a gold star icon displayed in front of their name.

You can also make modules ignore friends specifically, which can be changed in the [Teams Module](../../../modules/meowtils/teams/).

## Usage

```/meowfriend | /mf <player>``` - Add a player to the friends list.

```/meowunfriend | /muf <player>``` - Remove a player from the friends list.
