# **Safelist**

This allows you to add players to a safelist, normally because they are not a threat. Safelisted players will have a green checkmark icon displayed in front of their name. This is your own personal safelist and is stored in your game directory with other mod files.

If you are personally safelisted, your safelist checkmark icon will instead be blue.

## Usage

`/safelist | /sl <player>` - Add player to safelist.

`/unsafelist | /usl <player>` - Remove player from safelist.

`/safelist | /sl info <player>` - Shows if a player is safelisted.

!!! Example

    /sl tatp

## Automatically safelist

You can use the [AutoSafelist Module](../../../modules/meowtils/antisnipe/autosafelist/) to automatically add players that take a final death in Bedwars to the safelist.
