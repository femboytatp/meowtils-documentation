# **AutoSwap**

#### Automatically swaps items when about to run out.

!!! Warning

    This automatically swaps slots for you, which might be detected on advanced anticheats.

---

## Items

This will swap to the same item you last held, but if that no longer exists it will fall back to a whitelist that is shown below.

**Applies to:**

### Blocks

- Stone
- Grass
- Dirt
- Planks
- Wool
- Wood
- Glass
- Leaves
- Clay
- Logs

### Projectiles

- Egg
- Snowball

### Resources

- Iron Ingot
- Gold Ingot
- Diamond
- Emerald

### Pearls

- Ender Pearl

### Swords

- Swords

#### Swap on attack

Swaps to first available sword if you attack a player.

### Tools

- Fishing Rod
- Pickaxe
- Axe
- Shovel
- Hoe
- Flint and Steel
