# **SkywarsAlerts**

#### Alerts you when players hold important items.

---

## Alert

### Chat

Sends a chat message.

### Notification

Sends alert as a toast notification.

### All

Sends chat message & sends toast notification.

## Ping sound

### All

Plays a sound for all alerts.

### Important

Only plays a sound for important alerts.

**Applies to:**

- Ender Pearl
- Diamond Sword
- Knockback Rod
- Strength Potion

### None

Plays no sound.

## Cooldown

The cooldown in-between alerts for the same item, per player.

## Show distance

Also shows the distance to that player in the alert message.

## Items

Select which items to alert for.

## Nametag Icon

This will display an icon in player nametags if they ever held an important item this game.
