# **PearlDetector**

#### Detects thrown pearls and positions.

---

## Color

Change the color of HUD element, for your own timer.

## Text scale

Change text size for your own timer.

## Ping sound

Play a sound when a pearl is thrown.

## Timer for self

Displays a timer for yourself if you throw a time warp pearl, counting down until you get warped back.

## Position for others

Displays a box where someone will warp back to if they used a time warp pearl.

## Thrown pearl alert

Alerts in chat if someone throws a pearl, if that pearl type is selected.
