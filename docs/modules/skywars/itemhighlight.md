# **ItemHighlight**

#### Highlights important in your inventory by drawing a colored slot background.

---

## Opacity

Change the opacity of the background.

## Blacklist

Set the background color for blacklisted items.

## Safelist

Set the background color for safelisted items.

## Best

Set the background color for items that are considered best, this depends on currently equipped items.

## Prevent drop for best

Prevents you from dropping items that are considered best.

## Prevent drop for safelisted

Prevents you from dropping any items that are in the safelist.

## Render Blacklisted Items

Render background slot for blacklisted items.

## Render Safelisted Items

Render background slot for safelisted items.

## Render Best Items

Render background slot for items that are considered best.

## Items

Select which items to include when considering best items.

## Commands

!!! Note

    The item lists use item ID's, not item names. You can toggle displaying item ID's by enabling it in Minecraft settings or by doing F3 + H at the same time.

`/itemsl` - _Add an item to the item safelist._

`/itemusl` - _Remove an item from the item safelist._

`/itembl` - _Add an item to the item blacklist._

`/itemubl` - _Remove an item from the item blacklist._
