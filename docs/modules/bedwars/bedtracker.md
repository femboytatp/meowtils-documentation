# **BedTracker**

#### Tracks the location of your bed and alerts if an enemy approaches it.

---

## Text color

Changes the color for the HUD text.

## Scale

Changes text size for HUD.

## Alert frequency

The cooldown of alerts for each player.

## Max distance

The distance from bed where it starts to alert for enemy players.

## Ping sound

Plays a cat sound each alert.

## Show HUD

Displays HUD text showing you if you currently have a bed and how far away it is.

<span style="color: #aa0000">⚠</span> = Bed is out of render distance.
