# **EventTimers**

#### Displays timers for different events such as game events and emerald spawns.

---

## Scale

Size of the HUD element.

## Events

### Enabled

Enables game event timers.

### Show next events only

Only shows the next 2 events rather than all.

### Roman numerals

Replaces numbers with roman numerals. E.g. 3 becomes III.

### Dynamic color

Changes color for each event title.

## Emeralds

### Enabled

Enables emerald spawn timers.

### Dynamic color

Changes color depending on time left until spawn.
