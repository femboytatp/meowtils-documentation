# **ConsumeAlerts**

#### Alerts you when a player consumes a specific item.

---

## Alert

### Chat

Prints a chat message.

### Notification

Sends alert as a toast notification.

### All

Prints chat message & sends toast notification.

## Max distance

Sets the required distance, if a player is not within this distance it will not alert for them.

By default it is set to 0, which means it has no limit.

## Ping sound

Plays a sound for alerts.

## Show distance

Shows the distance to them in the alert.
