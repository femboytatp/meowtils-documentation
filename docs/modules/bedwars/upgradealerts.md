# **UpgradeAlerts**

#### Alerts when a team buys a team upgrade.

Applies to:

- Sharpness
- Protection

---

## Alert

### Chat

Sends a chat message.

### Notification

Sends alert as a toast notification.

### All

Sends chat message & sends toast notification.

## Ping sound

Plays a sound for alerts.
