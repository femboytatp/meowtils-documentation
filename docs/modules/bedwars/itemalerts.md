# **ItemAlerts**

#### Alerts you when players hold important items.

---

## Alert

### Chat

Prints a chat message.

### Notification

Sends alert as a toast notification.

### All

Prints chat message & sends toast notification.

## Ping sound

### All

Plays a sound for all alerts.

### Important

Only players a sound for important alerts.

### None

Plays no sound.

## Distance

### All

Shows distance to player for all items.

### Important

Only shows distance to player for important items.

**Applies to:**

- Potions
- Bridge Egg
- Diamond Sword
- Iron Golem
- Bedbug
- Charlie The Unicorn
- Diamond Pickaxe
- Enchanted Bow
- Milk
- Ender Pearl

### None

Shows no distance.

## Cooldown

Adjusts the cooldown for alerts per player.

## Items

Select which items you want to alert for.
