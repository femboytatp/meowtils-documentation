# **BedESP**

#### Renders an ESP for beds & obsidian.

---

## Color

Changes the color of the bed ESP box. Only applies to beds.

## Opacity

Changes the opacity of the ESP.

## Mode

### Full

Renders filled ESP boxes.

### Outline

Only renders outlined ESP boxes.

## Show obsidian

Shows obsidian that is placed around the bed.

## Bedwars only

Only renders ESP boxes if you are currently in a Bedwars game.
