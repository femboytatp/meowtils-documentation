# **BedESP**

#### Renders an ESP for beds & obsidian around them.

---

## Color

Changes the color of the bed ESP box.

## Opacity

Changes the opacity of the ESP.

## Mode

### Full

Renders a filled ESP boxes.

### Outline

Only renders outlined ESP boxes.

## Show obsidian

Shows obsidian that is placed around the bed.

## Bedwars only

Only renders ESP boxes if you are currently in a Bedwars game.
