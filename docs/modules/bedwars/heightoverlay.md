# **HeightOverlay**

#### Displays max height for Bedwars maps.

---

## Text color

Changes the color for HUD text.

## Show HUD

This enables a HUD text element that shows your current height & max height of current map.

## Scale

Size of HUD element.

## Show wool overlay

This visually recolors wool if it is placed at the max height.

## Color

Allows you to adjust the wool color if it's placed at max height.
