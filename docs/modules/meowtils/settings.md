# **Settings**

#### General mod settings.

---

## Auto-Updates

If enabled, new updates will automatically be installed. This is opt out and if disabled it will only provide a link to the update page for you to manually install it.

## Smooth font

Use smooth font for HUD elements and modules that render text in world. If disabled the normal Minecraft font will be shown instead.

This does not apply to GUI, which will always use smooth font.

## Copy chat

Lets you copy chat messages by hovering over them and right clicking.

!!! Bug "Important"

    Certain chat mods as well as Lunar Client may break this, which in that case you should use the copy chat feature provided by those mods instead.

## Prefix

**Modes:** _Default, Myau, Fire, Nebula, Air, Custom, Short_

Lets you customize the prefix for chat messages.

!!! Info
    
    Use the /theme command to pick colors for Custom & Short themes.

### Lowercase

Will make all prefix letters lowercase.

## Preview

Prints a message in chat to preview currently selected theme.
