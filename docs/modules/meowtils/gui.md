# GUI

#### Lets you adjust GUI related settings.

---

## Color

Adjusts the color theme that the Click GUI uses.

## GUI Scale

**Modes:** _Tiny, Small, Normal, Large, Huge_

Changes the scale for GUI, which may be necessary depending on monitor resolution.

## Reset GUI color

Resets the GUI color back to default theme color.

## Show tooltips

Shows a tooltip with a short description of what modules do when hovering them.

## Blur background

Applies a background blur while GUI is open.

## Bind

The bind for opening the GUI.
