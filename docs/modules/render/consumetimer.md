# **ConsumeTimer**

#### Displays a countdown of how much time left to consume an item.

---

## Text color

Changes the color for the HUD text.

## Scale

Changes size for HUD text.

## Mode

### Ticks

Displays countdown in ticks.

### Seconds

Displays countdown in seconds.

## Dynamic color

Changes color based on time left.
