# **Indicators**

### Highlights dangerous projectiles.

---

## Mode

### 3D

Renders a 3D ESP box.

### 2D

Renders a 2D ESP box.

## Render

### Full

Renders a filled ESP box.

### Outline

Renders an outlined ESP box.

## Opacity

Changes the opacity of the ESP.

### Expand

Normally the ESP uses the bounding boxes of the projectile, but expand lets you increase the size of it to be more visible. 

This just increases the ESP size not the actual bounding box of the projectile.
