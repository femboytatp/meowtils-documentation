# **ChestESP**

#### Renders an ESP on chests.

---

## Mode

### Opened

Only renders an ESP for chests that you opened.

### Normal

Renders an ESP for any chests.

### Both

Renders an ESP for any chests, as well as opened chests.

## Render

### Full

Renders a full ESP box.

### Outline

Renders outline for ESP box only.

## Chest color

Change color for chest ESP box.

## Opened color

Change ESP color for chests that have been opened.

## Skywars only

Only enables this module if you are in a Skywars game.
