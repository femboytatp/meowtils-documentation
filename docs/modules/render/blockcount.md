# **BlockCount**

#### Displays currently held block count and alerts when running low on blocks.

---

## Text color

Changes color of the HUD text.

## Scale

Changes text size.

## Alert threshold

Alerts you when you only have this many blocks left if alert when low is enabled.

## Alert when low

Toggle alerting if low on blocks.

## Ping sound

Plays a sound for alerts.
