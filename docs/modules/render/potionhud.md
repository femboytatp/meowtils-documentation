# **PotionHUD**

#### Displays important active potion effects on HUD.

---

## Name

### Full

Shows the full potion name.

### Short

Shows a short version of the potion name.

### None

Shows no potion name and instead colors the timer text to a color related to the specific potion.

## Display

### Both

Displays active potion effects in HUD and chat alerts when close to running out.

### HUD

Only displays active potions effects in HUD.

### Chat

Only alerts you when you are running out of a potion effect.

## Scale

Changes the size of the HUD text.

## Alert time

The time left for a potion effect at which it should alert you in chat that it is about to run out. If using 0 seconds this would alert you after the effect has already ran out.

## Expire sound

Plays a sound for alert.

## Hide infinite effects

Hides effects that you have infinite duration of from HUD.

## Potions

Select which potion effects to display.
