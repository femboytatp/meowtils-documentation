# **ItemScale**

#### Allows you to change the scale of dropped items.

---

## Scale

Changes the scale of dropped items.

## Important only

Only changes scale for important items.

**Applies to:**

Resources:

- Iron Ingot
- Gold Ingot
- Diamond
- Emerald
- Golden Apple
- Potions

Gear:

- Swords
- Pickaxes
- Axes
- Shovels
- Hoes
- Armor

Heads:

- Player Heads

## Include

Pick which type of items should be considered important.
