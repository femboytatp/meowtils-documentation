# **BreakProgress**

#### Renders a text while breaking blocks displaying how much time left until broken, or how far into the break progress you are.

---

## Mode

### Percentage

Shows your current break progress, 100% would be fully broken block.

### Time

Counts down until block is broken.

## Text color

The color of the break progress text.

## Text opacity

The opacity of the text.

## Scale

The size of the break progress text.

## Dynamic color

Will change color depending on how close you are to breaking the block.
