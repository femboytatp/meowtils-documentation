# **ItemESP**

#### Renders an ESP box and text for dropped items.

---

## Mode

### 3D

Renders a 3D ESP box.

### 2D

Renders a 2D ESP box.

## Render

### Full

Fills the entire ESP box.

### Outline

Only renders outline for the ESP box.

## Display

### Both

Displays both ESP box and text.

### Box

Only displays the ESP box.

### Text

Only displays the text.

## Font

### Normal

Renders the normal Minecraft font for text.

### Smooth

Renders the smooth Meowtils font for text.

## Text scale

Controls the size of the text.

## Box opacity

Changes the opacity of ESP boxes.

## Auto-scale

Automatically scales the size of the text and ESP box depending on how far away you are to the dropped item.

## Show stack size

Shows the current stack size of the item in the text.

## Show distance

Shows the current distance to the item from you.

## Compact text

If enabled, only displays a number showing stack size with no more information.

## Dynamic text color

Colors the text to a similar color of the item it is representing, rather than always being white.

## Bedwars only

Only enables this modules if you are currently in a Bedwars game.
