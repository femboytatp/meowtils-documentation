# **HotbarLock**

#### Allows you to lock slots in your hotbar which prevents you from dropping items in those slots.

---

## Mode

### Manual

Manually select which hotbar slots should be locked.

### Swords

Automatically lock any hotbar slots that has a sword.

## Show locked slots

Shows a red slot background for slots that are locked.

## Slots

Select which slots should be locked, only applies when using manual mode.
