# **NoParticles**

#### Removes certain particles to improve visibility and performance.

---

## Remove bedwars glyph

Removes the particles above a generator etc, creating a glyph. This significantly improved performance.

## Remove bedwars sponge

Removes all of the particles created when a sponge is placed in Bedwars.

## Remove block break

Removes all block destroy particles.
