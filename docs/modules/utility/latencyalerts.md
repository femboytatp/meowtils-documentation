# **LatencyAlerts**

#### Alerts you in chat for packet losses to the server. May not work properly on all servers.

---

## Alert

### Chat

Sends a chat message.

### Notification

Sends alert as a toast notification.

### All

Sends chat message & sends toast notification.

## Latency threshold

The packet loss in ms that is the threshold for an alert to be sent.

## Ignore limbo

Disables this module while you're in Hypixel limbo.

!!! Bug

    This only exists because you consistently get alerts while in Hypixel limbo. It is recommended that you keep this enabled.
