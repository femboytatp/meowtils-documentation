# **LatencyAlerts**

#### Alerts you in chat for packet losses to the server. May not work properly on all servers.

---

## Alert

### Chat

Prints a chat message.

### Notification

Sends alert as a toast notification.

### All

Prints chat message & sends toast notification.

## Latency threshold

The packet loss in milliseconds that is the threshold for an alert to be sent.

## Ignore limbo

Disables this module while you're in Hypixel limbo.
