# MurdererFinder

#### Resolves relevant roles in Murder Mystery.

---

## Mode

### 3D

Uses 3D mode for ESP.

### 2D

Uses 2D mode for ESP.

## Render

### Full

Renders a filled ESP box for relevant roles.

### Outline

Renders outlined ESP box for relevant roles.

### None

Renders no ESP at all for players.

## Name icons

Adds an icon for relevant roles to player names.

If they have a bow a golden arrow icon will be shown.

If they are the murderer red exclamation marks will be shown.

## Chat alerts

Alerts you in chat when a relevant role is found.

## Ping sound

Plays a sound if relevant role is found.

## Find murderers

Toggles checking for murderers.

## Find detectives

Toggles checking for detectives, which just checks if someone is holding a bow as it is not possible to differentiate between detectives and survivors that have a bow.
