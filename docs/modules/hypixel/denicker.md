# **Denicker**

#### Alerts you if someone is nicked and attempts to get their real username when possible.

---

## Nick message

Sends a message in chat if someone is nicked, and their username if it was possible to denick them.

## Nick tag

Will display a tag in a players nametag if they are nicked.

## Tag

### Normal

Uses the default icon.

### Asterisk

Uses an asterisk (*) as the icon.

### Text

Will use [NICKED] text instead of icons.

## Color

**Modes:** _Dark Purple, Dark Red, Dark Blue, Dark Aqua._

Lets you customize the color that nick icons/text will use.
