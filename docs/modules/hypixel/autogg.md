# AutoGG

#### Sends a custom message when a game ends, or before one starts.

---

## Auto GG

Sends set messages after a game ends.

### Enabled

Toggle sending message on game end.

### First message

The first message to send, if enabled.

### Delay

Will send this message after set delay.

### Second message

Will send a second message after the first one if enabled.

### Delay

Sends after set delay, which uses the same start time as first message.

!!! Example

    If both delays are set to 100ms, they will send at the same time.

## Auto GL

Sends a message before game starts.

### Enabled

Toggle sending message on game start.

### Send at

Will send the message when there is x seconds left until game start.
