# SessionStats

#### Display your own gamemode stats for the current instance or per game.

---

## Color

Change the color for the HUD element.

## Display

**Modes:** _Horizontal, Vertical_

Lets you adjust the way the HUD element is displayed.

## Reset

Reset current stats.

## Scale

Changes text size for the HUD element.

## Reset after game

If enabled it will automatically reset all of your stats after each game, rather than storing your stats for this session.

## Game recap message

Prints a chat message with the stats you got last game.

## Bedwars

Change if Bedwars session stats should be enabled and which stats to display.

## Skywars

Change if Skywars session stats should be enabled and which stats to display.
